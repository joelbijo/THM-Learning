# Active Directory

## Overview

**Active Directory** is a centralized system for managing users, computers, and security permissions across a network.

- Used for authentication and authorisation.

The server that runs the Active Directory services is known as a **Domain Controller (DC)**

**Active Directory Users and Computers**
Gives the hierarchy of users, computers and groups that exist in the domain.
