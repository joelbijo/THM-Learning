# Windows Fundamentals

## Computer Management (compmgmt) utility

### System tools

**Task Scheduler**
Used to create and manage common tasks that computer will carry out automatically at the times specified.

**Event Viewer**
Used to view events that have occurred on the computer.
It has 3 panes:

- The pane on the left provides a hierarchical tree listing of the event log providers.
- The pane in the middle displays a general overview and summary of the events specific to a selected provider.
- The pane on the right is the actions pane.

## Resource Monitor (resmon)

Resource Monitor displays per-process and aggregate CPU, memory, disk, & network usage information, and to provides details abt which processes are using individual file handles and modules.
